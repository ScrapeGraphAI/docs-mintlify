## 概述

ScrapeGraphAI API提供强大的端点，用于AI驱动的网页抓取和内容提取。我们的RESTful API允许您从任何网站提取结构化数据，处理本地HTML内容，并将网页转换为清晰的markdown格式。

## 认证

所有API请求都需要使用API密钥进行认证。您可以从[控制面板](https://dashboard.scrapegraphai.com)获取您的API密钥。

```bash
SGAI-APIKEY: your-api-key-here
```

<Note>
请保持您的API密钥安全，切勿在客户端代码中暴露它。使用环境变量来安全地管理您的密钥。
</Note>

## 基础URL

```bash
https://api.scrapegraphai.com/v1
```

## 错误处理

我们的API使用常规HTTP响应代码：

- `200` - 成功
- `400` - 错误请求
- `401` - 未授权
- `429` - 请求过多
- `500` - 服务器错误

<Note>
查看我们的[错误处理指南](/api-reference/errors)获取有关错误响应及其处理方法的详细信息。
</Note>
