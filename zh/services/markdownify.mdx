## 概述

Markdownify是我们的专业服务，可以将网页内容转换为清晰、格式良好的markdown。它能智能地保留内容结构，同时删除不必要的元素，非常适合内容迁移、文档创建和知识库构建。

<Note>
在我们的[交互式平台](https://dashboard.scrapegraphai.com/)上立即试用Markdownify - 无需编码！
</Note>

## 快速开始

### 快速入门

<CodeGroup>
```python Python
from scrapegraph_py import Client

client = Client(api_key="your-api-key")

response = client.markdownify(
    website_url="https://example.com/article"
)
```

```typescript TypeScript
import { Client } from 'scrapegraph-js';

// Initialize the client
const sgai_client = new Client("your-api-key");

try {
  const response = await sgai_client.markdownify({
    websiteUrl: "https://example.com/article"
  });

  console.log('Request ID:', response.requestId);
  console.log('Result:', response.result);
} catch (error) {
  console.error(error);
} finally {
  sgai_client.close();
}
```

```json
  "result": "# 理解AI驱动的网络爬虫\n\n随着AI技术的出现，网络爬虫已经发生了显著的演变...\n\n## 主要优势\n\n- 提高准确性\n- 智能提取\n- 结构化输出\n\n![AI爬虫流程](https://example.com/images/ai-scraping.png)\n\n> AI驱动的爬虫代表着网络数据提取的未来。\n\n### 入门指南\n\n1. 选择目标网站\n2. 定义提取目标\n3. 选择合适的工具\n",
  "error": ""
}
```

响应包括：
- `request_id`：用于追踪请求的唯一标识符
- `status`：转换的当前状态
- `result`：包含markdown内容和元数据的对象
- `error`：错误信息（如果在转换过程中出现任何错误）
</Accordion>

### 异步支持

对于需要异步执行的应用程序，Markdownify通过`AsyncClient`提供异步支持：

```python
from scrapegraph_py import AsyncClient
import asyncio

async def main():
    async with AsyncClient(api_key="your-api-key") as client:
        response = await client.markdownify(
            website_url="https://example.com/article"
        )
        print(response)

# 运行异步函数
asyncio.run(main())
```
