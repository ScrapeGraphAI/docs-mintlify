## 概述

LocalScraper提供与SmartScraper相同的强大AI提取功能，但专门用于处理本地HTML内容。这使其特别适用于已有HTML内容或需要处理缓存页面、内部文档或动态生成内容的场景。

```typescript JavaScript
import { Client } from 'scrapegraph-js';
// Initialize the client
const sgai_client = new Client("sgai-6a2...220f00e9");

// Your HTML content
const htmlContent = `
<html>
    <body>
        <h1>ScrapeGraphAI</h1>
        <div class="description">
            <p>AI-powered web scraping for modern applications.</p>
        </div>
    </body>

try {

    userPrompt: "Extract the company description and contact information",
    websiteHtml: htmlContent

  console.log('Result:', response.result);
} catch (error) {
  console.error(error);
} finally {
  sgai_client.close();
}
```

## 高级用法

### 自定义模式示例

精确定义您想要提取的数据：

<CodeGroup>

```python Python
from pydantic import BaseModel, Field
from typing import List

class ProductData(BaseModel):
    name: str = Field(description="产品名称")
    price: str = Field(description="产品价格")
    description: str = Field(description="产品描述")
    specifications: List[str] = Field(description="产品规格")

response = client.localscraper(
    website_html=html_content,
    user_prompt="提取产品信息",
    output_schema=ProductData
)
```

```typescript TypeScript
import { z } from 'zod';

const ProductSchema = z.object({
  name: z.string().describe('产品名称'),
  price: z.string().describe('产品价格'),
  description: z.string().describe('产品描述'),
  specifications: z.array(z.string()).describe('产品规格')
});

const response = await localScraper(
  apiKey,
  html_content,
  '提取产品信息',
  ProductSchema
);
```

### 异步支持

对于需要异步执行的应用程序，LocalScraper通过`AsyncClient`提供异步支持：

```python
from scrapegraph_py import AsyncClient
import asyncio

async def main():
    html_content = """
    <html>
        <body>
            <h1>产品：游戏笔记本</h1>
            <div class="price">¥6999.99</div>
            <div class="description">
                配备RTX 3080的高性能游戏笔记本。
            </div>
        </body>
    </html>
    """
    
    async with AsyncClient(api_key="your-api-key") as client:
        response = await client.localscraper(
            website_html=html_content,
            user_prompt="提取产品信息"
        )
        print(response)

# 运行异步函数
asyncio.run(main())
```
