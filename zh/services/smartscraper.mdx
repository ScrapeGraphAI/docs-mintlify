## 概述

SmartScraper是我们的旗舰LLM驱动的网络爬虫服务，可以智能地从任何网站提取结构化数据。通过使用先进的LLM模型，它能像人类一样理解上下文和内容，使网络数据提取比以往更加可靠和高效。

## 快速开始

### 快速入门

<CodeGroup>
```python Python
from scrapegraph_py import Client

client = Client(api_key="your-api-key")

response = client.smartscraper(
    website_url="https://scrapegraphai.com/",
    user_prompt="Extract info about the company"
)
```

```typescript TypeScript
import { Client } from 'scrapegraph-js';
// Initialize the client
const sgai_client = new Client("sgai-6a2...220f00e9");

try {
  const response = await sgai_client.smartscraper({
    websiteUrl: "https://scrapegraphai.com/",
    userPrompt: "Extract info about the company"
  });

  console.log('Request ID:', response.requestId);
  console.log('Result:', response.result);
} catch (error) {
  console.error(error);
} finally {
  sgai_client.close();
}
```

## 高级用法

### 自定义模式示例

精确定义您想要提取的数据：

<CodeGroup>

```python Python
from pydantic import BaseModel, Field

class ArticleData(BaseModel):
    title: str = Field(description="文章标题")
    author: str = Field(description="作者姓名")
    content: str = Field(description="文章主要内容")
    publish_date: str = Field(description="发布日期")

response = client.smartscraper(
    website_url="https://example.com/article",
    user_prompt="提取文章信息",
    output_schema=ArticleData
)
```

```typescript TypeScript
import { z } from 'zod';

const ArticleSchema = z.object({
  title: z.string().describe('文章标题'),
  author: z.string().describe('作者姓名'),
  content: z.string().describe('文章主要内容'),
  publishDate: z.string().describe('发布日期')
});

const response = await smartScraper(
  apiKey,
  'https://example.com/article',
  '提取文章信息',
  ArticleSchema
);
```

```curl curl
curl -X 'POST' \
  'https://api.scrapegraphai.com/v1/smartscraper' \
  -H 'accept: application/json' \
  -H 'SGAI-APIKEY: your-api-key' \
  -H 'Content-Type: application/json' \
  -d '{
  "website_url": "https://example.com/article",
  "user_prompt": "提取文章信息",
  "output_schema": {
    "type": "object",
    "properties": {
      "title": {
        "type": "string",
        "description": "文章标题"
      },
      "author": {
        "type": "string",
        "description": "作者姓名"
      },
      "content": {
        "type": "string",
        "description": "文章主要内容"
      },
      "publish_date": {
        "type": "string",
        "description": "发布日期"
      }
    },
    "required": ["title", "author", "content", "publish_date"]
  }
}'
```
</CodeGroup>

### 异步支持

对于需要异步执行的应用程序，SmartScraper通过`AsyncClient`提供异步支持：

```python
from scrapegraph_py import AsyncClient
import asyncio

async def main():
    async with AsyncClient(api_key="your-api-key") as client:
        response = await client.smartscraper(
            website_url="https://example.com",
            user_prompt="提取主要内容"
        )
        print(response)

# 运行异步函数
asyncio.run(main())
```
