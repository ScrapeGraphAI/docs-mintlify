## 概述

[CrewAI](https://github.com/joaomdmoura/crewAI)是一个用于编排角色扮演AI代理的框架。通过Scrapegraph CrewAI集成，您可以轻松地将网络爬虫功能整合到您的代理工作流程中。

## 安装

安装所需的包：

```bash
pip install crewai scrapegraph-tools python-dotenv
```

## 可用工具

### ScrapegraphScrapeTool

`ScrapegraphScrapeTool`为您的CrewAI代理提供网络爬虫功能：

```python
from crewai import Agent, Crew, Task
from crewai_tools import ScrapegraphScrapeTool
from dotenv import load_dotenv

# 初始化工具
tool = ScrapegraphScrapeTool()

# 使用该工具创建代理
agent = Agent(
    role="网络研究员",
    goal="从网站研究和提取准确信息",
    backstory="您是一位专业的网络研究员，在从各种网站提取和分析信息方面经验丰富。",
    tools=[tool],
)
```

<Accordion title="完整示例" icon="code">
```python
from crewai import Agent, Crew, Task
from crewai_tools import ScrapegraphScrapeTool
from dotenv import load_dotenv

# 加载环境变量
load_dotenv()

# 初始化Scrapegraph工具
tool = ScrapegraphScrapeTool()

# 使用Scrapegraph工具创建代理
agent = Agent(
    role="网络研究员",
    goal="从网站研究和提取准确信息",
    backstory="您是一位专业的网络研究员，在从各种网站提取和分析信息方面经验丰富。",
    tools=[tool],
)

# 为代理定义任务
task = Task(
    name="爬取任务",
    description="访问网站https://scrapegraphai.com并提取有关创始人的详细信息，包括他们的姓名、角色和任何相关背景信息。",
    expected_output="包含从网站提取的信息的文件。",
    agent=agent,
)

# 使用代理和任务创建团队
crew = Crew(
    agents=[agent],
    tasks=[task],
)

# 执行任务
result = crew.kickoff()
```
</Accordion>

## 配置

在您的环境中设置Scrapegraph API密钥：

```bash
export SCRAPEGRAPH_API_KEY="your-api-key-here"
```

或使用`.env`文件：

```bash
SCRAPEGRAPH_API_KEY=your_api_key_here
```

<Note>
从[控制面板](https://dashboard.scrapegraphai.com)获取您的API密钥
</Note>
