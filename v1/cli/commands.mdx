## Available Commands

| Command | Description |
|---------|-------------|
| `sgai smartscraper` | Extract data from a webpage using AI |
| `sgai searchscraper` | Search and extract from multiple sources |
| `sgai markdownify` | Convert webpage to markdown |
| `sgai credits` | Check remaining API credits |
